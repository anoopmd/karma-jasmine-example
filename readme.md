# Karma Jasmine Example

A Simple Karma Jasmine Example

## Install

```sh
$ yarn install
$ bower install
```

## Launch Karma
```sh
$ node_modules/karma/bin/karma start
```

## Run Jasmine in browser
```sh
open jasmine-spec-runner.html in your browser
```
